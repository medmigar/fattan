Créer son site Web

Objectifs de ce cours

Bienvenue sur ce site qui a pour objectif d'annoncer le début d'un cours qui vous apprendra à faire un site web professionel. Cette offre annoncera l'institut dans laquelle sera donnée ce cours et la date de son commencement.
Créer un site web c'est passionnant (مثير ), il n'y a pas de limite pour votre imagination. Le temps nécessaire dépendra de vos attentes et de vos goûts, mais je peux vous donner les moyens techniques pour mettre en ligne un site internet basique, à travers des cours, pour les débutants sur HTML5, CSS3 et JavaScript. L'objectif est de modifier les fichiers et de comprendre comment tout cela fonctionne pour adapter le contenu à vos besoins.
Dans ce cours, vous allez:

  • Créer une page Web de base en HTML
  • Appliquer des styles aux éléments de la page à l'aide de CSS
  • Créer des thèmes en utilisant CSS
  • Ajouer une interactivité avec l'utilisateur via le JavaScript
  • Exécuterer le site dans votre navigateur par défaut
  • En fin inspecter le site Web à l'aide des outils de développement des navigateurs

Un site web, c'est quoi ?

Un site web c'est un ensemble de fichiers codés avec les langages HTML,CSS et JavaScript, accessible depuis une adresse web. Il peut être:
statique, c'est-à-dire qu'il possède un contenu fixe ou dynamique comme une eboutique.
La différence entre ces deux modèles est l'interactivité avec l'utilisateur, technologie à connaître. Un site Statique est codé seulement avec HTML et CSS, tandis que le site Dynamique en plus de HTML et CSS, il est lié à un serveur: l'ordinateur distant qui répond à la demande de la page. On peut utiliser un serveur local, sur notre propre ordinateur, pour faire des tests avant de mettre le travail sur un serveur distant public.
Dans ce cours, on va créer une structure de projet simple qui comprend trois fichiers: un fichier HTML, un fichier CSS et un fichier JavaScript.
Le language HTML sert à décrire la structure de la page web que le navigateur doit afficher. HTML se compose d'une série d'éléments:les balises HTML qui indiquent au navigateur comment afficher le contenu. Le navigateur n'affiche pas les balises HTML, mais il les utilise pour déterminer comment afficher le contenu du document. Depuis les débuts du World Wide Web, il existe de nombreuses versions de HTML. Dans ce cours on va suivre la dernière norme HTML5.
Remarque: Un site web n'est pas forcément accessible depuis l'internet. Il peut l'être uniquement en local, sur sa propre machine, ou sur un réseau privé. On parle d'intranet. Cette solution existe surtout pour les entreprises. Il permet une plus grande sécurité et ainsi, on bloque toute agression extérieure au réseau, et on facilite les raisons de performance et de maintenance.
