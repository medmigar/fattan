Introduction

Le CSS est le langage utilisé pour styliser des pages Web. Il permet de définir le style des éléments HTML, et la mise en page et les variations d'affichage pour différents appareils et tailles d'écran. Il existe plusieurs types de propriétés disponibles pour styliser les éléments HTML. Ici, ce générateur de code est destiné à créer les bordures d'un div de cinq façons différentes. Sachant qu'il existe trois façons d'insérer une feuille de style dans un document HTML:

  • CSS externe
  • CSS interne
  • CSS en ligne
