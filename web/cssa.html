Le CSS أوراق الأنماط المتتالية

مقدمة

في الأيام الأولى للويب، لم يكن CSS موجودًا. وكان في بداية الويب، لغة HTML فقط، والتي ظهرت في عام 1991. وCSS هي اللغة التي جاءت لتكملة HTML في عام 1996. وتسمح بالتنسيق، وتزيين النتيجة المرئية النهائية، كتحديد اللون والحجم والمحاذاة من النص إلخ…. وCSS هي اختصار العبارة: Cascading Style Sheets التي يمكن ترجمتها إلى: أوراق الأنماط المتتالية، ونقول "Style sheet" لأننا بشكل عام نكتب كود CSS في ملف منفصل. يحتاج CSS إلى صفحة HTML للعمل. وإن ملف HTML عند فتحه، يحمّل ملف CSS ويطبق معلومات أسلوبه.

  • CSS هي لغة تصف نمط مستند HTML، وهي اللغة المستخدمة لتصميم مستند HTML.
  • CSS يصف كيفية عرض عناصر HTML على الشاشة أو في الطباعة أو في بعض الوسائط الأخرى.
  • CSS يوفر الكثير من العمل. يمكنه التحكم في تخطيط صفحات الويب المتعددة في نفس الوقت.
  • يتم تخزين أوراق الأنماط الخارجية في ملفات CSS.
  • باستخدام ملف ورقة نمط خارجي، يمكن للمرء تغيير مظهر موقع ويب بأكمله عن طريق تغيير ملف واحد.
  • CSS يسمح بتنويعات التخطيط والعرض لمختلف الأجهزة وأحجام الشاشات.
  • CSS يتيح تحديد الخط المستخدم على الموقع، كما يسمح لك بتحديد حجم النص والحدود والخلفية إلخ….
  • CSS يتمم كود HTML وينسقها ويعطي محتواها الشكل المراد فعله.
  • عندما يقرأ المستعرض (Navigateur) ورقة الأنماط (CSS)، فإنه يقوم بتنسيق مستند HTML بناءً على المعلومات الموجودة في ورقة الأنماط.

أين نكتب CSS؟

    لدينا خيار كتابة وإدراج ورقة الأنماط في الكود في ثلاثة أماكن مختلفة:
  • في ملف ورقة الأنماط الخارجية مستند CSS خارج مستند HTML وهي أكثر الطرق الموصى بها وتسمى CSS خارجي.
      <head>
        <link rel="stylesheet" href="style.css">
      </head>
  • في رأس < head > لملف HTML (CSS داخلي) إذا كان لصفحة HTML واحدة نمط فريد. يمكن استخدام ورقة الأنماط الداخلية إذا كان للصفحة الواحدة نمط فريد. يتم تحديد الأنماط الداخلية في عنصر < style >، في قسم < head > بصفحة HTML. 
  <head>
    <style>
      body {
        background-color: linen;
      }
      h1 {
        color: maroon;
        margin-left: 40px;
      } 
    </style>
  </head>
  • مباشرة في عنصر HTML عبر السمة: style (أسلوب CSS الأقل الموصى به على الإنترنت). يمكن استخدام النمط المضمن لتطبيق نمط فريد على عنصر HTML واحد.. لاستخدام الأنماط المضمنة، نضف سمة النمط إلى العنصر المناسب. يمكن أن تحتوي سمة النمط على أي خاصية CSS. يوضح المثال أدناه كيفية تغيير اللون والهامش الأيسر لعنصر < h1 > 
  <h1 style="color:blue;margin-left:30px;">This is a heading</h1>

يجب أن تتضمن كل صفحة مرجعًا لملف ورقة الأنماط الخارجية داخل عنصر < link >. يدخل عنصر < link > داخل قسم < head >. يمكن كتابة ورقة أنماط خارجية في أي محرر نصوص. يجب ألا يحتوي الملف على أي علامات HTML. يجب حفظ ملف ورقة الأنماط بامتداد .css.
