Le CSS compléte le langage HTML et mettre en forme la page web. Il existe de nombreuses façons de sélectionner la portion de la page que l'on veut mettre en forme. on peut viser:
toutes les balises d'un même type, en écrivant simplement leur nom.
certaines balises spécifiques, auxquelles on a donné des noms à l'aide des attributs class ou id.
uniquement les balises qui se trouvent à l'intérieur d'autres balises. etc.
langage à balise se traduit par markup language.
Valideur: Le W3C fournit un service de validation en ligne de balisage: http://validator.w3.org/
l’indentation pour faciliter la lecture.
Comme pour le HTML, le W3C fournit un service en ligne de validation de CSS: http://jigsaw.w3.org/css-validator/
