le Javascript

Introduction

Le Javascript est un langage de programmation de scripts orienté objet. Il permet aux développeurs d'écrire du code source. Le code source est un ensemble d'actions, appelées instructions. Elles donnent des ordres à l'ordinateur afin de faire fonctionner le programme.
En fonction du code source, l'ordinateur exécute différentes actions. Il existe trois manières d'utiliser du code source:
On donne le code source à un programme appelé compilateur pour le lire et le convertir dans un langage que l'ordinateur sera capable d'interpréter: le langage binaire, qui est fait de 0 et de 1.
le code source est compilé partiellement, dans un code plus simple à lire pour l'ordinateur, mais qui n'est pas encore du binaire. Ce code intermédiaire devra être lu par ce que l'on appelle: «machine virtuelle», qui exécutera ce code. On l'appelle: Langage précompilé.
Le code source reste tel quel, et si on veut exécuter ce code, on doit le fournir à un interpréteur qui se chargera de le lire et de réaliser les actions demandées. On l'appelle: Langage interprété.
Les scripts sont majoritairement interprétés. Et quand on dit que le Javascript est un langage de scripts, cela signifie qu'il s'agit d'un langage interprété! Il est donc nécessaire de posséder un interpréteur pour faire fonctionner du code Javascript, et l'interpréteur qu'on utilise est fréquemment inclus dans des navigateurs Web!
Chaque navigateur possède son propre interpréteur Javascript, qui diffère selon les navigateurs. Pour Internet Explorer, son interpréteur Javascript s'appelle JScript celui de la version 9 s'appelle Chakra, celui de Mozilla Firefox se nomme SpiderMonkey et celui de Google Chrome est V8.
Le Javascript est à ce jour utilisé majoritairement sur Internet, conjointement avec les pages Web. Il permet de dynamiser des pages HTML, en ajoutant des interactions, des animations avec l'utilisateur.
Le Javascript est un langage dit client-side, c'est-à-dire que les scripts sont exécutés par le navigateur chez l'internaute (le client). Cela diffère des langages de scripts dits server-side qui sont exécutés par le serveur Web. C'est le cas des langages comme le PHP, Java...
Un script server-side va s'occuper de « créer » la page Web qui sera envoyée au navigateur. Ce dernier va alors afficher la page puis exécuter les scripts client-side tel que le Javascript. Le Javascript a évolué vers d'autres destinées. Il est utilisé pour réaliser des extensions des différents programmes, et utilisé pour réaliser des applications. Le Javascript est utilisé majoritairement au sein des pages Web.
